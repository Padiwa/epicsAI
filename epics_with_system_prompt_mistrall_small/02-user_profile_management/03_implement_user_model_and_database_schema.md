# Implement User Model and Database Schema

**Description:**
As a developer, I need to create a User model and database schema to store user information.

**Go-Specific Considerations:**
- Use GORM for ORM and database interactions.
- Define the User struct with fields like ID, Username, Email, PasswordHash, and CreatedAt.

**Acceptance Criteria:**
- User model is defined with necessary fields.
- Database schema is created and migrated.
- User data can be saved and retrieved from the database.
