# Implement User Profile Retrieval

**Description:**
As a user, I want to view my profile information so that I can see my details.

**Go-Specific Considerations:**
- Use GORM to query the database for user data.
- Implement middleware for authentication.

**Acceptance Criteria:**
- Users can retrieve their profile information.
- Profile data is returned in JSON format.
- Only authenticated users can access their profile.
