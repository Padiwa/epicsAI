# Implement User Deletion

**Description:**
As a user, I want to delete my account so that I can remove my data from the application.

**Go-Specific Considerations:**
- Use GORM to delete user data from the database.
- Implement middleware for authentication.

**Acceptance Criteria:**
- Users can delete their account.
- User data is removed from the database.
- Only authenticated users can delete their account.
- Deletion endpoint returns a success message.
