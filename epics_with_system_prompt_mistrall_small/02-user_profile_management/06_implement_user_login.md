# Implement User Login

**Description:**
As a user, I want to log in to my account so that I can access my profile and shared workouts.

**Go-Specific Considerations:**
- Use bcrypt to verify password hashes.
- Implement JWT for session management.

**Acceptance Criteria:**
- Users can log in with their email and password.
- Password verification is successful.
- JWT tokens are generated and returned upon successful login.
- Login endpoint returns a success message.
