# Create User Profile API Endpoint
**Description:** Develop an API endpoint to retrieve user profile information for authenticated users.
**Go-Specific Considerations:** Use the Chi router for handling HTTP requests. Ensure the endpoint is protected with JWT authentication middleware.
**Acceptance Criteria:**
- API endpoint for user profile retrieval is implemented.
- Endpoint is protected with JWT authentication.
- Endpoint returns user profile information.
- Endpoint returns appropriate success and error responses.
